DreamTome is a beautifully crafted, thematic dream journaling application designed to be your private and offline-first companion for exploring the world of your subconscious. With a unique, book-like interface and AI-powered features, DreamTome transforms the act of recording dreams into a magical experience.

## ğŸ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

-   [Node.js](https://nodejs.org/en/) (v18 or higher)
-   [pnpm](https://pnpm.io/) (or npm/yarn)

### Installation

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/harryneopotter/DreamTome.git
    cd DreamTome
    ```

2.  **Install dependencies:**
    ```sh
    pnpm install
    ```

### Running the Application

1.  **Start the development server:**
    ```sh
    pnpm dev
    ```

2.  **Open your browser:**
    Navigate to `http://localhost:5173` (or the address shown in your terminal).
